== Description ==

[youtube http://www.youtube.com/watch?v=xsMm6ATJf0Y]


= English =
If you are using a great WooCommerce plugin for your web store and you need to upload an image of a large variety of products - this plugin is for you!
How to do it?
1. Activate the plugin.
2. Open a list of products or categories.
3. In the area where there should be a thumbnail, drag an image file (the area where you want to throw the file in the right moment will be highlighted).
4. Wait for the image.
5. Done.

= Русский =
Если вы используете отличный плагин WooCommerce для своего интернет магазина и Вам нужно загрузить изображения большого множества товаров - этот плагин для Вас!
Как это сделать?
1. Активируйте плагин.
2. Откройте список товаров или категорий.
3. В область, где должна быть миниатюра, перетащите файл изображения (область, куда нужно скинуть файл в нужный момент будет подсвечена).
4. Дождитесь загрузки изображения.
5. Готово.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload folder `hiweb-wc-thumbnail-upload` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'hiWeb WooCommerce Thumbnails Upload' menu in WordPress
